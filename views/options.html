ZAP Browser Extension

This extension gives ZAP access to the browser.
These options should be set correctly if the browser is launched from ZAP but may need to be set manually if you install this add-on in a browser not launched by ZAP.
For example http://zap/ or http://localhost:8080/
As set in the ZAP API options
For more info see https://github.com/zaproxy/browser-extension/
