AccuKnox DAST Browser Extension

This extension gives AccuKnox DAST access to the browser for security testing.
These options should be set correctly if the browser is launched from DAST but may need to be set manually if you install this add-on in a browser not launched by DAST.
For example http://zap/ or http://localhost:8080/
As set in the DAST API options
For more info see https://www.accuknox.com/
